Instructions

Every piece of code begins with an IP address that holds the clue to the next step.

Example:

011.658.625.23:/hello there player
011.658.625.23:/welcome to my game

First highlighted number from each pair represents the row, second highlighted number represents the word.
(Only words count. All other characters are ignored).

Result from the example above:

11 = first row, first word = hello

23 = second row, third word = game

Result => :/hello game (NOTE! Every line in the console begins with :/ )

In the bottom right corner you will find two squares. The first one will tell you which server to select from the dropdown list.
The colored one will tell you what IP address to select during the game.

When all settings are done, press enter in the console and move on to the next step.

Recommended resolution: 1920x1080
